The user enters the BowersWorld.com webpage via Google Pages.
Selects Visit Our Library button.
Has option to sign in or register.
If already registered the user enters the user ID which is his email address and a password.
The system validates the user by checking the Google drive user database and if they are valid are allowed entry to the Library application. Otherwise input a message is generated.
If the user selects to register they will be presented with questions to answer.
Username
Email address 
First name 
Last name 
Zip code 
Agree to terms (must). Terms will be available for display.
A message will be displayed notifying them of need to authentic and offer options:
Email 
Text
Google 
GitHub 
Facebook
Microsoft 
The validation process will take place.
The user will be advised to watch for email and approve registration before access to the Library will be permitted.
The validation process within the system:
Capture log in data if successful:
User email address 
Date and time in Google database 
If new user registration capture registration data wether successfully completed process or not with date and time.
When the user enters the Library system a welcome message will be displayed with daily librarian message and user status and any personal messages .
The user settings are downloaded from the user database defining any rate limits or other user data.
The system will check the Library database version and if mandatory download is required a new version will be downloaded. If the database is out of date the user will be offered the opportunity to update his database.
The user will use the various options to find books in the database and will be able to display and download copies according to his stored settings.
Book titles , authorb and other data like thumbnails are in the database. The ebooks are located on the Google drive.
Any queries made by the user will be saved in the database with user ID and time and data stamps.
